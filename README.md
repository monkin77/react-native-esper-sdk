# react-native-esper-sdk

React Native Module that interacts with the Esper SDK

## Installation

```sh
npm install react-native-esper-sdk
```

## Usage

```js
import { multiply } from "react-native-esper-sdk";

// ...

const result = await multiply(3, 7);
```
